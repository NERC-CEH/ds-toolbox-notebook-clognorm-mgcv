# ds-toolbox-notebook-clognorm-mgcv

This vignette describes how to fit censored log-normal models in `mgcv`.

You will need to install the [`mgcvUtils`](https://github.com/dill/mgcvUtils) package which includes the necessary extensions to the `mgcv` package. You can install this from R using the following:

```r
remotes::install_github("dill/mgcvUtils")
```
